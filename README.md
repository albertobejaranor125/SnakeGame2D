# SnakeGame2D

A 2D arcade Snake game built in Unity, based on the open-source Zigurous project tutorial, featuring custom gameplay modifications and system refactoring.

## 🎮 Gameplay Overview
The player controls a snake to consume food spawning on the grid. Each piece of food increases the snake's length and awards points. The game tracks scores and features real-time pause states.

## 🕹️ Controls
* **WASD** : Move the snake grid-by-grid
* **Esc** : Pause / Resume the gameplay loop
